📌 프로젝트 소개

유튜브 크리에이터를 위한 AI 기반 채널 성장 전략 플랫폼

CRiT는 유튜브 채널 데이터를 분석하여 맞춤형 콘텐츠 주제, 대본, 성장 인사이트를 제공. 데이터 분석부터 AI 추천, 성과 진단까지 한 번에 처리하여 크리에이터의 더 나은 기획과 성장 지원.

🎯 주요 기능

01 영상 추천

  • 채널 성향과 입력 키워드를 기반으로 AI가 맞춤형 콘텐츠 주제 3가지 추천
  • 선택한 주제에 대한 타겟 시청자, 예상 조회수, 상세 기획안 제공
  • AI 기반 제목 5개 및 썸네일 가이드 자동 생성

크리에이터가 다음 영상을 기획할 때 필요한 모든 정보를 AI가 한 번에 제공.

02 채널 분석

  • 유튜브 데이터를 분석하여 채널 성장 상태와 핵심 지표 시각화
  • 도달, 참여, 유지, 성장 4가지 지표 기반 종합 점수 산출
  • AI 요약 리포트로 채널의 강점과 개선 방향 제시

별도 입력 없이 로그인만으로 내 채널 현황을 객관적 데이터로 확인.

03 영상 상세 분석

  • 채널의 개별 영상 단위 성과를 분석하여 AI 기반 피드백과 개선 방향 제공
  • 도달률, 시청 지속 시간, 추천 확장성 등 핵심 지표 분석
  • 동일 카테고리 내 백분위 비교로 상대적 성과 확인

영상별 성과를 데이터로 분석하여 다음 영상의 개선 포인트 도출.

04 트렌드 분석

  • 현재 YouTube에서 인기 있는 키워드를 워드클라우드로 시각화
  • 키워드 클릭 시 해당 주제로 영상 추천 기능 연결
  • 실시간 트렌드 파악으로 시의성 있는 콘텐츠 기획 지원

트렌드 키워드를 통해 현재 시청자들이 관심 있는 주제 파악.

📊 백분위 기반 점수 모델

296,000개 이상의 실제 유튜브 영상 데이터(14개 카테고리, 6,100+ 채널)를 수집·분석하여 구축한 데이터 기반 점수 모델.

점수 공식

점수 = VPS 백분위 × 0.40 + 채널평균대비 백분위 × 0.40 + 일평균조회수 백분위 × 0.20
요인 가중치 설명
VPS (views_per_sub) 40% 구독자 대비 조회수
채널 평균 대비 40% 채널 평소 성적 대비 상대 성과
일평균 조회수 20% 업로드 후 경과일 대비 조회수

비교 그룹

  • 구독자 구간: S(~50K), M(50K~200K), L(200K~500K), XL(500K+)
  • 영상 유형: 숏폼(60초 이하) / 롱폼
  • 카테고리: 14개 카테고리별 분리 비교

🎬 서비스 흐름 - 유저 온보딩

1

로그인 페이지 진입

Google OAuth를 통한 간편 로그인 제공. 로그인 시 사용자의 YouTube 채널 정보 자동 연동.

2

메인 페이지 이동

로그인 완료 후 메인 대시보드로 자동 리다이렉트. 트렌드 키워드 워드클라우드와 주요 기능 바로가기 제공.

3

영상 추천

키워드 입력 후 AI가 다음 영상 주제 3가지 추천. 선택한 주제에 대한 상세 기획안 제공.

4

채널 분석 자동 실행

연동된 채널 데이터를 기반으로 종합 점수 및 성장 단계 분석. 별도 입력 없이 바로 내 채널 현황 확인 가능.

5

영상 상세 분석

채널 분석 페이지에서 개별 영상 클릭 시 상세 분석 페이지로 이동. 도달률, 시청 지속 시간, 추천 확장성 분석 및 AI 개선점 제시.

🔄 UX 플로우 - 기능 간 연결

사용자가 자연스럽게 다음 행동으로 이어질 수 있도록 기능 간 연결 설계.

메인 페이지→영상 추천워드클라우드 키워드 클릭 시 해당 주제로 영상 추천 페이지 바로 연결
채널 분석→영상 상세 분석영상 목록에서 개별 영상 클릭 시 상세 분석 페이지 이동
영상 상세 분석→영상 추천분석 결과 기반 해당 카테고리의 다음 영상 추천 기능 연결
'분석 → 개선 → 기획'의 순환 구조로 별도 탐색 없이 현재 맥락에서 다음 단계로 자연스럽게 이동 가능.
